# 📰 CDEK News — Тестовое задание

Тестовое задание для стажировки Frontend-разработчик в СДЭК.

## 🚀 Демо

🔗 **[Посмотреть работающую версию](https://cdek-news-internship.vercel.app)**)

## 🛠 Технологический стек

- **React 18** + **TypeScript** — надёжность и типизация
- **Vite** — быстрая сборка и разработка
- **Tailwind CSS** — современная стилизация
- **Tabler Icons** — иконки (используются в продуктах СДЭК)
- **react-intersection-observer** — ленивая загрузка изображений

## ✨ Реализованные функции

### Основные требования
- ✅ Загрузка новостей с тестового API СДЭК
- ✅ Отображение скелетона при загрузке
- ✅ Плейсхолдер для пустого состояния
- ✅ Пагинация с кнопками "Назад"/"Вперёд"
- ✅ Адаптивность (mobile-first подход)
- ✅ Обработка ошибок API

### Дополнительные возможности
- ✅ **Полная типизация TypeScript** — все интерфейсы и типы
- ✅ **Кэширование запросов** — загруженные страницы хранятся 5 минут
- ✅ **Ленивая загрузка изображений** — через IntersectionObserver
- ✅ **Плавные анимации** — появление карточек и переходы страниц
- ✅ **Форматирование дат** — "Сегодня", "Вчера", "5 дней назад"
- ✅ **Форматирование чисел** — 1.5K, 2.3M для счётчиков

## 📦 Установка и запуск

```bash
# Клонирование репозитория
git clone https://github.com/your-username/cdek-news-test.git

# Переход в директорию
cd cdek-news-test

# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр собранного проекта
npm run preview
